# University Timetable Webpage



A simple HTML and CSS project that recreates a university timetable interface. The webpage displays a weekly class schedule with a navigation bar, course selection menus, and color-coded timetable entries.



### Features



\- Weekly timetable layout (Monday–Friday)

\- Navigation bar with university logo

\- Degree, year, and group selection menus

\- Color-coded subjects for better readability

\- External links to the university website and student/teacher portals

\- Simple and responsive table layout using HTML and CSS



### Technologies Used



\- HTML5

\- CSS3



### Project Structure



```

project/

```



### How to Run



1\. Download or clone this repository.

2\. Open `index.html` in any modern web browser.

3\. No installation or additional software is required.
